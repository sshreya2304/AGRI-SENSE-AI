# 🌾 Agri Sense AI

Agri Sense AI is an AI-powered smart agriculture platform designed to help farmers make informed decisions through crop recommendations, weather forecasting, disease detection, government scheme information, and an intelligent chatbot.

## 🚀 Features

### 🌱 Crop Recommendation
- AI-based crop suggestions
- Helps farmers choose suitable crops
- Supports better yield planning

### ☁️ Weather Forecast
- Real-time weather information
- Temperature and humidity updates
- Helps farmers plan agricultural activities

### 🤖 AI Chatbot
- Agriculture-related assistance
- Farmer-friendly guidance
- Quick answers to farming queries

### 🦠 Disease Detection
- Detect crop diseases
- Provides early awareness
- Helps reduce crop losses

### 🏛️ Government Schemes
- Information about farmer welfare schemes
- Subsidy and support program details
- Easy access to agricultural benefits

### 👨‍🌾 Farmer Community
- Connect with other farmers
- Share knowledge and experiences
- Promote collaborative farming practices

### 👤 User Authentication
- Login and Signup functionality
- Secure user access
- Personalized user experience

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/AGRI-SENSE-AI.git
```

### 2. Navigate to Project

```bash
cd AGRI-SENSE-AI
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start Development Server

```bash
npm run dev
```

### 5. Open Browser

```text
http://localhost:5173
```
